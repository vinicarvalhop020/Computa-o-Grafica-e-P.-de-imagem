# Laboratório de Introdução à Computação Gráfica e Processamento de Imagens

Bem-vindo ao repositório de códigos desenvolvidos durante as aulas de laboratório da disciplina de **Introdução à Computação Gráfica e Processamento de Imagens**! Aqui você encontrará implementações de algoritmos, técnicas e conceitos fundamentais dessas áreas.

## 📋 Sobre o Repositório

Este repositório contém códigos desenvolvidos durante as sessões de laboratório da disciplina. Os programas abordam tópicos como manipulação de pixels, conversão de espaços de cores, filtros de imagem, transformações geométricas e muito mais. O objetivo é fornecer uma base prática para entender os conceitos teóricos da computação gráfica e do processamento de imagens.
